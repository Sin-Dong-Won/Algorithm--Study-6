NP 문제 집합

P  문제 집합 NP 문제 집합을 둘다 포함하는 문제이다.

NP 문제는 비결정적 다항식 시간 알고리즘이다.
즉, 해를 찾는 알고리즘이 아닌
다항식 시간에 확인하는 문제

P 문제를 해결하는데 다항식 시간이 걸리므로 NP 알고리즘과 대응 가능

P 문제를 위한 NP 앍리즘은 해를 추측하는 단계를 생략하고, 해를 확인하는 단계 대신에 해를 직접 다항식 시간에 구하고
확인 결과를 '맞다'라고 답한다.

NP 알고리즘은 추측한 해를 확인하여 맞다/아니다 라고 답하므로, 문제의 해가 예 아니오가 되도록 주어진 문제를 변형해야한다.

NP 완전 문제의 특성을 알기 위해 어떤 문제를 다른 문제로 변환하는 과정을 이해해야한다.


문제의 변환: 문제 A를 해결하기 위해 문제 B를 해결하는 알고리즘을 이용하는 것을 의미

 문제 A의 입력을 문제 B의 입력 형태로 변환시키고,  B의 해에서 A의 해를 변환시키는 것
